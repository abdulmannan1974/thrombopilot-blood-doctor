# ThromboPilot by Blood Doctor

Responsive clinical decision-support app for anticoagulation, thrombosis risk scoring, perioperative planning, renal dosing, and acute thrombosis workflows.

## Features

- Interactive pathways for perioperative anticoagulant planning, AF dosing, thrombophilia testing, and VITT screening
- Bedside calculators for PE, DVT, AF, bleeding risk, ACS, cancer-associated thrombosis, renal dosing, and TIA assessment
- Mobile, tablet, and desktop-friendly interface

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
